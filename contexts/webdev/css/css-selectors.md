### Structure

An template for two CSS selectors and their properties:&#x20;

```css
(selector1) {
    (property1) : (value1) 
}

(selector2) {
    (property2) : (value2)
}

...
```

For example, to make a link red with no underline:

```css
a {
    color: red;
    text-decoration: none;
}
```
